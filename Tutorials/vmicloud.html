How To Access Remote Services

IMPORTANT

Remote use is only available to cadets wih faculty permission and must be approved by IT

Accessing vmiedu.cloud.com#

  1. Install Citrix Workspace App from https://www.citrix.com/downloads/workspace-app/. If you need help installing please see Citrix's documentation

  2. Go to http://vmiedu.cloud.com

  3. Log in using your VMI email address and password

  4. Click Log On

  1. Select Detect Workspace
    • Depending on what browser you are using you may get a popup asking to open the Citrix Receiver Launcher.
    • Click to checkmark "Always allow vmiedu.cloud.com to open links of this type in the associated app then click Open Citrix Receiver Launcher

  1. Once you are logged into https://vmiedu.cloud.com, select Apps and then select All Apps on left menu to expand the list of available Apps. From this menu you can select the App you wish to launch.

Example: How to use Remote Desktop App#

  1. From the All Apps menu select Remote Desktop Connection

  1. If prompted, select Do not ask me again for this site and then select Permit use.

  1. Remote Desktop Connection popup will come up. Enter your computer name of the computer you are wanting to remote into. Select Connect

  1. Enter your VMI login credentials

Access files from M: Drive#

IMPORTANT

If you only need to access files from your M: Drive you can use the Windows 10 Desktop Virtual Environment without needing remote access to a physical asset

  1. Select Desktops and then select All Desktops on the left menu to expand the list. From this menu select Windows 10 Desktop

  1. This will launch a new Windows 10 Desktop - Desktop Viewer window. You may see a connecting loading screen. Please wait for this to connect.

  1. Read the the Warning! You are accessing the Virginia Military Institute (VMI) System and hit OK

  1. Once Windows has loaded and logged you in, you can access your M: drive through file explorer
