Many of our software vendors offer educational/academic software packages that are free for use for Engineering students and faculty.  Below you will find a list of all the software currently available for downloads as a student/cadet or faculty of the Engineering departments of VMI.

Solidworks

PLEASE READ INSTRUCTIONS CAREFULLY

IMPORTANT PREREQUISITES 

- This is for Windows Operating System only.  This is NOT SUPPORTED BY SOLIDWORKS FOR OSX (Mac)
- 20 GB disk space free, download of Solidworks will fail if space is not available

- If you DO NOT have (a) previous version(s) of Solidworks installed please skip the "Uninstall previous versions of Solidworks" section and proceed to "Download and install Solidworks Instructions."

- If you DO have (a) previous version(s) of Solidworks you MUST uninstall any prior versions of Solidworks before proceeding.  Be sure to do a complete uninstall.

UNINSTALL PREVIOUS VERSIONS OF SOLIDWORKS WITH A COMPLETE UNINSTALL

Please note the following when uninstalling: backup all previous Solidworks Data you want to keep, and note that you can choose to perform a standard, custom, or complete uninstall.

The standard uninstall removes Program Files and Folders associated with Solidworks products.  By default, all Solidworks products for the selected release are specified for a standard uninstall.

A custom uninstall allows you to choose and removes one or more of the following: Program Files and Folders, registry keys and data folders (Solidworks Toolbox & files and folders from the original download locations..)

A complete uninstall removes the installatoin directories, registry keys, and data folders

1.  Press the Windows Key (⊞) + R to bring up the run command prompt/textfield.
2.  Type "appwiz.cpl" and hit enter to bring up the "Programs and features" to uninstall software.  Scroll down and select the previous version(s) of Solidworks you are uninstalling.
3.  On the uninstall screen, ensure that the product(s) you want to uninstall is selected (If you uninstall all products, the Solidworks Installation Manager is also uninstalled).
4.  Clear the products you do not want to remove, if you retain any product(s), Solidworks Installation Manager is not removed
Choose whether to perform a custom or complete uninstall (Complete uninstalls are recommended just make sure you have any Solidworks data backed up beforehand). 
5.  Click change in advanced options, the Advanced Options screen appears. The program Files and Folders and Folders option is checked by default.  Select all options to perform a complete uninstall. 
6.  Click Back to Summary.
7.  On the Summary screen, expand Advanced Options to display the uninstall method and entries 
8.  Click Remove Items 

DOWNLOAD AND INSTALL SOLIDWORKS INSTRUCTIONS

1.  Goto: http://www.solidworks.com/sw/education/SDL_form.html
2.  Fill out the form to request the software and license 
3.  Select "No" for "I already have a Serial Number that starts with 9020:" 
    - Please contact your professor or Luke McMasters (mcmasterslj@vmi.edu) for the current years school ID number 
    - Select the newest version 
4.  This will bring you to the terms and conditions page, please read and click Accept Agreement 
5.  You will receive an email from Solidworks with a serial number and expiration date (license good for 365 days), keep track of this as you will need it during the installation process. Make sure to use the CAD serial number 
6.  At the top of the next page there will be a link for Download which will download the web installer/installation manager 
7.  Run the SolidWorksSetup.exe and allow it to self-extract and bring up the installation
8.  Follow the on screen installation directions which will guide you through the complete download and installation process 

Cadets who are having issues downloading or installing the product please goto the Barracks Help Desk in New Barracks room C01-B.  If you are an engineering cadet you can also see the Divisional Engineering IT Specialist in NEB 406.
