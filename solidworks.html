SOLIDWORKS

PLEASE READ INSTRUCTIONS CAREFULLY

Important Prerequisites

  • This is for Windows Operating Systems only and OSX  (Mac) is NOT officially supported by Solidworks
  • If you have any previous versions of Solidworks installed please see below as you MUST uninstall any prior versions of Solidworks before proceeding.
  • If you DO NOT have previous versions of Solidworks installed or are doing a fresh install please skip the uninstall steps and scroll down to Download and Install Solidworks Instructions or click here.

Uninstall Previous Versions of Solidworks

Please note the following when uninstalling: backup all previous Solidworks Data you want to keep, and note that you can choose to perform a standard, custom, or complete uninstall.

The standard uninstall removes Program Files and Folders associated with Solidworks products. By default, all Solidworks products for the selected release are specified for a standard uninstall.

A custom uninstall allows you to choose and removes one or more of the following: Program Files and Folders, registry keys and data folders (Solidworks Toolbox & files and folders from the original download locations..)

[RECOMMENDED] A complete uninstall removes the installatoin directories, registry keys (expired serial numbers), and data folders

  1. Press the Windows Key (⊞) + R to bring up the run command prompt/textfield
  2. Type "appwiz.cpl" and hit enter to bring up the "Programs and features" to uninstall software. Scroll down and select the previous version(s) of Solidworks you are uninstalling
  3. On the uninstall screen, ensure that the product(s) you want to uninstall is selected (If you uninstall all products, the Solidworks Installation Manager is also uninstalled
  4.  Clear the products you do not want to remove, if you retain any product(s), Solidworks Installation Manager is not removed.  Choose whether to perform a standard, custom, or complete uninstall (Complete uninstalls are recommended just make sure you have any Solidworks data backed up beforehand)
  5. Click change in advanced options, the Advanced Options screen appears. The program Files and Folders and Folders option is checked by default. Select all options to perform a complete uninstall
  6. Click Back to Summary
  7. On the Summary screen, expand Advanced Options to display the uninstall method and entries
  8. Click Remove Items 

Download and Install Solidworks Instructions

Please email Luke McMasters (MCMASTERSLJ@VMI.EDU) to request the current years School ID Number. You will need this before being able to download the Solidworks Installer. If you encounter installation errors due to the network issues with the downloader you can additionally access \\syscenter10\Software_Installs\SOLIDWORKS 2020 SP03 while connected to the VMI domain to copy the already downloaded installation files to your hard drive and use that instead of the downloader as the installation instructions are still the same.

1. Goto: https://www.solidworks.com/support/community-download#no-back.  Fill out all the information under Contact Information and make sure to use your VMI email.  Under Product Information Select "No" for "I already have a Serial Number that starts with 9020, this is where you will input the School ID that you requested from Luke McMasters.  Select the newest version under Academic Version then click Request Download.
2. This will bring you to the terms and conditions page, please read and click Accept and Continue.
3. This next page will show your SOLIDWORKS CAD Serial Number.  Please take note of this and either take a picture, print, copy, or write this number down as you will need this to install SOLIDWORKS.  Please note this license is good for up to a year.  Each year you will need to get a new license.
4. Click the Download button to download the SolidWorksSetup.exe file then run it to allow it to self extract to start the SOLIDWORKS Installer.
5. If you see this message at the start of the setup, please restart your computer then restart the setup.
6. Select "Install on this computer" and click next.
7. Please input your SOLIDWORKS CAD Serial Number from Step 3 (Do not use the one in the screenshot as it is a dummy placeholder serial number and will not work).
8. Read and Check the "I accept the terms of the SOLIDWORKS License Agreement, then click Download and Install
9. Please stay connected to the internet as the setup is downloading and installing components at the same time.  The time is dependent on your internet speeds.  Do not disconnect from the internet during this installation.  Once complete if it asks to restart/reboot, then restart/reboot the computer then open Solidworks and verify it opens and works properly.
