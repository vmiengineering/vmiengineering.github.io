Mathcad 15 M045 Installation 

Download Mathcad 15 M045 or access from Domain \\syscenter10\Software_Installs\Mathcad 15 M045

Before Installing (.NET 3.5 Prerequisite):

  • Unzip the MathCAD-Students.zip to any location, it will maintain the folder structure, then goto that folder
  • Make sure .NET 3.5 is enabled.  Press the Windows Key + R then type in "optionalfeatures" and hit enter.  Make sure the .NET Framework 3.5 (includes .NET2.0 and 3.0) checkbox is checked then hit OK.  You may be prompted to restart the computer, if so click button to restart.  If it is already checked/enabled skip to the next step (Install MSCML 4.0 SP2 Parser and SDK).

    DO THESE STEPS ONLY IF YOU HAVE PROBLEMS TRYING TO DOWNLOAD AND INSTALL THE .NET 3.5 FEATURE.

    1. Restart the computer then go to: Prerequisites folder
    2. Run ".NET 3.5 Framework Installer.bat"
    3. Right click the Windows start menu in the lower left hand corner and select "Command Prompt (Admin) if on Windows 10.  If on Windows 7 click the Windows menu in the lower left hand corner and search for "cmd" then right click and choose "Run as Administrator"
    4. Copy the following command in the cmd window and hit enter then wait for the process to complete: 
    DISM /Online /Enable-Feature /FeatureName:NetFx3 /All /LimitAccess /Source:C:\sources\

Installation Steps for the Ansys Student Products from Release 18.0 Onward:

Before Installing (MSCML 4.0 SP2 Parser and SDK Prerequisite):

  1. Go to: Prerequisites folder and open mscml.msi
  2. Click Next
  3. Accept ToS and click next
  4. Click Next
  5. Click Install Now

Install MathCAD 15 M045:

  1. Run: MathCAD 15 M045\MED -060602 –CD-150_M045\setup.exe
  2. Click Next
  3. Select “I Accept” and click next
  4. Click MathCAD
  5. Make sure Trial/New License is selected and hit next
  6. Select “Typical” and click next
  7. Towards the end of the installation a MathCad License Setup popup will come up. Select “I want to acquire a license via the internet” and click next.
  8. Select “Educational – Student”
  9. Enter Product Code send in email
  10. If you do not have a PTC Account click “Create Account” and follow the directions on creating a PTC account, then go back to the setup and hit next. (If you already have a PTC account you can just click next)
  11. Enter your PTC account credentials and click next
  12. Once you get the prompt that the MathCAD installation is complete you can click the exit button.

Mathcad 15 30 Day Free Trial

PTC offers Mathcad 15.0 for a free 30 day trial.  The trial version is the same as the full commercial version of PTC Mathcad with the except that it will expire 30 days after it is installed.

Trial Overview

PTC Mathcad allows you to experience enhanced calculation power with faster, more accurate solutions and comprehensive worksheets. PTC Mathcad’s ease-of use gets you started quickly, but its powerful features and units-aware math notation let you create and document complex calculations that can be verified and shared with colleagues.

Experience the Power of PTC Mathcad:

  • Easy to use – natural math notation means no special programming
  • Develop worksheets that are presentable and understandable
  • Improve verification & validation of critical calculations
  • Document worksheets containing calculations, graphs and text for sharing and reuse

Download

Go to https://www.mathcad.com/en/try-and-buy/free-trial and fill out the form using your VMI email then click submit. Then click the DOWNLOAD NOW button to download the installation files.

Installation (These instructions are provided on the next page after you sign up)

  1. Save the installation file to your computer (it slightly less than 300 MB, so it may take a few minutes for the download to complete depending on your Internet connection and computer performance)
  2. Right-click the Mathcad 15_EN.zip icon and choose "Extract All." Select a destination for the folder that you can easily locate (such as your desktop). This will create a folder at the specified destination called "Mathcad 15_EN"
  3. Open the Mathcad 15_EN folder and double-click the "setup.exe" file to start the installation
  4. Follow the instructions and on the PTC Mathcad License Type page, choose "Acquire new license" and continue with the installation
  5. At the end of the installation process, you will be guided to set up a license. Choose "I want to acquire a license via the Internet" and "I want a trial license" on the first two pages of the PTC Mathcad License Setup. You will need the trial license installed and activated to perform any worksheet conversions to Prime 3.0.

Mathcad Express - Free for Life

PTC offers free-for-life Engineering Math Software. PTC Mathcad Express is free math software that gives lifetime access to selected features of PTC Mathcad Prime 6.0

When you download PTC Mathcad Express and choose the 30-day full functionality option, you’ll get access to the full version of PTC Mathcad Prime 6.0 for 30 days. You’ll be able to:

  • Convert your units automatically to prevent costly errors
  • Analyze your results with interactive 2D and 3D plots
  • Document using standard math notation and advanced math formatting
  • Integrate your PTC Mathcad worksheet directly into a Creo model
  • Allows you to present your calculations with plots, graphs, text, and images in a single document
  • *At the end of 30 days, you’ll then automatically have lifetime access to PTC Mathcad Express, a lighter version of PTC Mathcad 6.0.

Download

*NOTE* Mathcad Express after the 30 trial may not have all the features that are available in Mathcad 15.0 M045 (Version used in labs). Please refer to the comparison chart to see the differences in features.

Go to https://www.mathcad.com/en/try-and-buy/mathcad-express-free-download and fill out the form using your VMI email and click submit.  You should receive an email from PTC with your trial download within two hours.  Please check your junk or spam email folders as well.  If you do not receive the email you will need to contact PCT at mathcadna@ptc.com.

Within the email will be provided a download link to download PTC Mathcad Express, click the "INSTALL SOFTWARE" button to download the installer.  Click the link for the installation guide provided on the same page as the download link to view and follow installation instructions.

Installation Quick Overview:

  • INTRODUCTION: Keep default "Install New Software" and click next
  • SOFTWARE AGREEMENT: Accept the software license agreement and click next
  • APPLICATION SELECTION: Keep the default application selections and click install
  • INSTALLATION: Wait until installation is complete, you will see progress indicators for estimate time left 
  • FINISH: Click Finish to exit the PTC Installation Assistant
