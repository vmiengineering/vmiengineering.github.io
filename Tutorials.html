How to Map Printers

Remove Old Printer

1.  Click the start menu button in the lower left hand corner of your screen
2.  Type "Devices and Printers"
3.  Make a note of the name of the printer that you are currently connected to
4.  Right click on your printer and select Remove Device
5.  Select Yes when prompted "Are you sure you want to remove this device?"

Map New Printer

1.  Click the start menu button in the lower left hand corner of your screen
2.  Type \\printservers and hit enter to open the print server
3.  Scroll through the list of printers and find the printer you wish to map
4.  Double-click on the printer to install

Setting Printer as Default

1.  Click the start menu button in the lower left hand corner of your screen 
2.  Type "Devices and Printers"
3.  Right click on the printer you want as your default printer and select "Select as Default Printer"
