ANSYS

Ansys offers educational student versions of their software.  Ansys Student is their Ansys Workbench-based bundle of Ansys Mechanical, Ansys CFD, Ansys Autodyn, Ansys SpaceClaim, and Ansys DesignXplorer.  Ansys Student is used by hundreds of thousands of students globally.  It is a great choice if you are already familiar with the Ansys workbench platform.

Student Education Version Installation

Before Installing:

  • Ensure that your computer meets the technical requirements necessary to run the software.  Please see the platform support document for details 

    Quick System Requirement overview:
    Windows 10 64bit
    Processor: Workstation Class
    4 GB RAM
    25 GB HD Space
    Physical C:/ Drive must be present
    GPU/Driver: Professional workstation class 3-D
    OpenGL Capable
  • Ensure that you have administrator privileges for the computer on which you are installing this product
  • Uninstall any existing Ansys products present on your computer.  The Ansys Student Installation will overwrite any existing installations
  • Please visit the Ansys Student Community for additional assistance

Installation Steps for the Ansys Student Products from Release 18.0 Onward:

  1. Goto https://www.ansys.com/academic/free-student-products and click the ANSYS Student picture and then click DOWNLOAD ANSYS STUDENT 202x Rx button.
  2. Extract (unzip) the downloaded installation files
  3. Right-click on setup.exe and select Run as Administrator. (This will run setup.exe from the the extracted files.)
  4. Read and accept the clickwrap to continue
  5. Click the right arrow button to accept the default values throughout the installation
  6. Click the exit button to close the installer
  7. The Ansys Student software is now installed. Congratulations!
  8. Reboot your machine and then run the Ansys Student product form your Start menu by selecting Workbench 18.0 (Or corresponding version number)
  9. To view System and Browser Requirements, Problem Size Limits, License Duration, What's Included, and Prior Release information click the corresponding tab links on the same page
