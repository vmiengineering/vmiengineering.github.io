EES (Engineering Equation Solver)

EES must be used while on the VMI Domain otherwise you will get a licensing error.  So please be either physically connected via Ethernet while on post, or connected to the Barracks-Secure or VMI-Sentinel wireless.

EES IS FOR WINDOWS OPERATING SYSTEMS ONLY.

Installation

  • You must be on the vmi domain to access the share below
  • Navigate to \\syscenter10\Software_Installs\EES 2020 10.835 and copy\download the following files to your asset:
  • EES.dft (this is the license file)
  • ees_manual.pdf (manual for EES)
  • Install Instructions Students.docx (same installation instructions in document format)
  • setup_ees_acad.exe (this is the installation executable file used to install EES)

1. Run setup_ees_acad.exe and click Next on the Welcome splash screen

2. Read the EULA and select the I Agree button then click next

3. Read the Installation Notes then click Next

4. Click next to keep the destination directory as the default

5. Keep all the defaults here and click Next

6. At the Ready to Install screen click Next

7. After installation completes click Finish

8. Navigate to where you downloaded the EES.dft license file.  Right click EES.dft file then click copy

9. Then Navigate to C:\EES32\ then right click and click Paste to copy the license file over

10. Verify EES.dft file got copied to C:\EES32.  It should like like in the screenshot

11. Open EES program and on startup you will see a splash popup like in the screen shot.  Click continue

12. On the top menu bar click Help then click License Information

13. Verify that the Academic Expiration Date reads as in the screenshot (2099-01-01).  You have completed the installation and verification of license expiration date.
