Solidworks

  • *PREREQUISITES BEFORE PROCEEDING WITH THE DOWNLOAD AND INSTALLATION*

    • This is for Windows Operating systems only, this is incompatible with OSX (MAC)
    • Be sure you have at least 20 GB of disk space free for download and installation. Download of SOlidworks will fail if space is not available
    • If you do not have previous versions of Solidworks installed please move on to the Download and Install Solidworks 2018 EDU/Community edition instructions:
    • If you have a previous version(s) of Solidworks on your machine be sure to backup your Solidworks data, then uninstall any prior versions of Solidworks before proceeding. Previous versions of Solidworks iwll not work after installing the EDU/Community version. Be sure to do a complete uninstall
      • UNINSTALL PREVIOUS VERSIONS OF SOLIDWORKS

        • Please note the following when uninstalling: You can choose to perform a standard, custom, or complete uninstall.

        The standard uninstall removes Program Files and Folders associated with a Solidworks product. By default, all Solidworks products for the selected release are specified for a standard uninstall

        A custom uninstall allows you to choose and removes one or more of the following items: Program Files and Folders, registry keys and data folders, such as the Solidworks Toolbox, and files and folders from the original download location

        A complete uninstall removes the installation directories, registry keys, and data folders

      1. Press the Windows Key + R to bring up the run command prompt/textfield
      2. Type "appwiz.cpl" and hit enter, this will bring up the "Programs and features" to uninstall software. Scroll down and select the previous version(s) of Solidworks you are uninstalling
      3. On the uninstall screen, ensure that the product you want to uninstall is selected (If you uninstall all products, the Solidworks Installation Manager is also uninstalled)
      4. Clear the products you do not want to remove, if you retain any product, Solidworks Installation Manager is not removed
      5. Choose whether to perform a custom or complete uninstall (Complete uninstalls are recommended just make sure you have any Solidworks data backed up beforehand)
        • Click change in advanced options, the Advanced Options screen appears. The program Files and Folders and Folders option is checked by default.
        • Select one or more options to perform a custom uninstall, or select all options to perform a complete uninstall
        • Click Back to Summary
        • On the Summar screen, expand Advanced Options to display the uninstall method and entries
      6. Click Remove Items

  • Download and Install Solidworks 2018 EDU/Community edition instructions:

    • Goto: http://www.solidworks.com/sw/education/SDL_form.html
    • Fill out the form to request the software and serial number
      • Select No for I already have a Serial Number that starts with 9020:
      • Input the school license ID of 92016SDK in the text field where it states If No, please enter SDK-ID, or Trial Code:
      • Select the version you want form the Version dropdown selection box
    • This will bring you to the terms and conditions page, please read and click Accept Agreement
    • You should receive an email from Solidworks with a serial number, keep track of this as you will need it during the installation process. Make sure ot use the CAD serial number
    • At the top of the next page there will be a link for Download which will download the web installer/installation manager
      • Run the SOlidWorksSetup.exe and allow ot self-extract and bring up the installation
    • Follow the on screen installation directions which will guide you through the complete download and installation process
  • Cadets who are having issues downloading or installing the product please goto the Barrakcs Help Desk in New Barrakc room C01-B
