Installation

Before Installing:

  • Ensure that your computer meets the technical requirements necessary to run the software.  Please see the platform support document for details 

    Quick System Requirement overview:
    Windows 10 64bit (not compatible with OSX)
    Processor: Workstation Class
    4 GB RAM
    25 GB HD Space
    Physical C:/ Drive must be present
    GPU/Driver: Professional workstation class 3-D
    OpenGL Capable
  • Ensure that you have administrator privileges for the computer on which you are installing this product
  • Uninstall any existing Ansys products present on your computer.  

Installation Steps for Ansys 2020R1

  1. Go to \\syscenter10\Software_Installs\ANSYS 2020 R1\Extracted while on the VMI Domain (Physically connected with ethernet or connected to Barracks-Secure or VMI-Sentinel.  You can copy and paste this into the file explorer text field or the Windows 10 search field
  2. Copy all the setup files to your local hard drive
  3. Right-click on setup.exe and select Run as Administrator
  4. Follow the installation video provided by ANSYS to complete the setup below.  Afterwards continue the directions below the video

Please note when it comes to the “Enter License Server Specification” screen at time stamp 1:12, select the button at the bottom and make sure “Skip this step and configure later” is check marked. DO NOT OPEN ANSYS AFTER INSTALLATION IS COMPLETE
