VMI's Education Institutional Agreement with ESRI allows VMI to download and use ArcGIS for academic use.

Installation Instructions‚Äã

  1. Run ArcGIS_Desktop_1082_180378.exe
  2. Click next to extract the temporary setup files and keep Launch the setup program checked and hit close to run the setup
  3. Click Next at the welcome screen
  4. Read the Master Agreement and select I accept the master agreement and click Next
  5. Keep Complete selected for installation type then click Next
  6. Keep Destinatino Folder to default and click Next
  7. Keep Python installation destination folder to default and click Next
  8. Click Install at the Ready to Install the Program screen
  9. When installation is done click Finish and the ArcGIS administrator Wizard will auto launch.
  10. See below for License Activation Instructions

Authorization License Activation‚Äã

  1. Under 1. Select a product - ArcGIS Desktop
    • Select Advanced (ArcInfo) Single Use
  2. Under 2. Launch the Authorization Wizard for Single Use products
    • Click Authorize Now
    • Click OK
  3. Under Authorization Options
    • Select I have received an authorization file and am now ready to finish the authorization process
    • Click Browse.. and select where you downloaded EducationSiteArcGISDesktop_SingleUse_1173428.prvc license file
    • Click Next
  4. Under Authorization Method
    • Select the default Authorize with Esri now using hte Internet. (This automatic method is the easiest way to authorize. It requires and Internet connection.)
    • Then click Next
  5. Under Authorization Information this information is automatically pulled from the license file.
    • Click Next
  6. Under Authorization Information (Continued) Please choose the best selection for your application. For cadets select the following:
    • Your Organization: Education-Student
    • Your Industry: Higher Education
    • Yourself: Student
  7. Under Software Authorization Number - ArcGIS Desktop Advanced (Single Use)
    • The authorization numbner will autofill from the license here. (If not input the following Authorization Number ESU244207723)
    • Click Next
  8. Under Authorize Software Extensions
    • Keep the default I have authorization number(s) to authorize one or more extensions
    • Click Next
  9. Under Evaluate Software Extensions
    • Leave defaults
    • Click Next
  10. Once software is authorized you should see the following text: Congratulations, your software has been authorized and is not ready for use
    • Click Finish
  11. ArcGIS Administrator will open
    • Click OK to close.
    • Installation and activation is now complete
