Autodesk

Autodesk (Autocad | Civil 3D | Revit | Forma)

Named User Licensing‚Äã

Autodesk has changed to Named User Licensing away from our Concurrent Floating License. We no longer have the option to self-host the Autodesk license on our license server

What Does This Mean?‚Äã

When a user first starts any Autodesk product on any computer lab machine they will be prompted to sign in using their Autodesk account, the computer no longer auto pulls a license from our licensing server.

FORMA Link‚Äã

Forma is a web-based Autodesk product which can be accessed here: Autodesk Forma | Forma Login

Computer Lab Use‚Äã

To access Autodesk products in the computer labs you will need to have an Autodesk account to login.

Already have an Autodesk Account linked to your VMI Email?‚Äã

  • Skip to step: Logging into Autodesk Software (If you have an autodesk account linked to a personal email it will not work in the computer labs. You will need to do the account creation process shown below)

Create Autodesk Account Instructions‚Äã

  • You will be sent an automated email invitation from Autodesk to create an Autodesk account with the subject Welcome to Autodesk
  • Click Create your account from the email link
  • Enter your First Name | Last Name | Confirm Email | Password
  • Agree to Autodesk's Terms of Use and Privacy Statement then click Create Account
  • You will see an Account Created window and when you click Done it will redirect you to your Autodesk Profile page which shows information about all your Autodesk products and licenses.

Logging into Autodesk Software‚Äã

  • Open any Autodesk software on a lab computer (Autocad, Civil 3D, Revit)
  • A licensing popup will come up and auto open a browser window to Autodesk login page
  • Sign in using your login credentials and select Stay Signed in
  • Check your email for a verification code, copy and paste/input the code then select Don't ask again on this browser for next 30 days and click Enter code
  • Record and\or download your backup code and select I have saved this code for future use. and click Continue
  • Once you are signed in click Go to product
  • Another popup will come up named Open Autodesk Identity Manager
    • Select Always allow accounts.autodesk.com to open links of this type in the associated app
    • Click Open Autodesk Identity Manager
  • This will take you back to the Autodesk product and you should see a brief popup in the upper right-hand corner with your email name that states Sign In Complete

Personal Use‚Äã

  1. Download acad.zip installer here
  2. Extract acad.zip to c:\acad (It must be extracted here)
  3. Right click and select Run as administrator c:\acad\Install Autocad 2026 [2025-2026 Academic Year - Named User License].bat
  4. A CMD window will open up and the silent installer will begin. Please do not do anything until you see the CMD window close.
  5. Autocad in now installed and can be logged into with your Autodesk Account.
