Ansys

Web Shared Named User Licensing‚Äã

  • VMI has changed over to Web Shared Licensing away from our Concurrent Floating License.

  • To access ANSYS in the computer labs you will need to have an Ansys account to login.

What does this mean?‚Äã

When a user first starts Ansys on any computer lab machine they will be prompted to sign in using their Ansys account, the computer no longer autopulls a license from our licensing server.

This now also opens up access to installing Ansys on your personal machines and being able to use it anywhere as long as you have access to the internet to reach the cloud licensing server.

Create Ansys Account Instructions‚Äã

REMINDER

Please note for testing purposes a non VMI email was used for the video tutorial, you will use your VMI email

  1. You will recieve an email to create an ansys account from Ansys ID with the subject You have been added to an Ansys account
  2. Click Click here to register button link from the email
  3. A internet browser window will open up to Ansys account creation site. Enter your VMI email
  4. Click Send verification code
  5. Open your email and look for an email from Ansys ID with the subject Ansys Account email verification code
    • Copy the verification code from the email
    • Paste the verification code into the text box named Verification code and click Verify code
    • Click Continue once your Email address is verified
  6. Input the requested information to create your account and make sure to Select I agree to Ansys User Terms of Use and ackowledge the Privacy Notice then click Create account button
  7. The next screen will let you choose which authentication method you would like to use. Select I want to use email verification only then click Continue
    • If you are familiar and confortable with Authenticator Apps you may choose that option instead
  8. You have completed creating your Ansys ID Account, this will be the account you use to open Ansys in the labs and on personal machines.

Computer Lab Use‚Äã

TIP

After the first sign on the same computer you will not need to sign in again. You will only need to sign in on a new computer you have not used before

Logging into Ansys on Lab Machines‚Äã

  1. Open any Ansys program (fluent will be used for this example)
  2. Once loaded it will auto open an internet browser window to Ansys' login page.
    • Input your VMI email address
    • Input your password
  3. Click Send verification code
  4. Check your email for the Ansys verification code and copy it then paste it into the verification code textbox and click Verify code
  5. Once your email address is verified click Continue
  6. Once you see You are now signed into your Ansys account. You can close this browser\tab and continue using the application
  7. You may encounter an error like in the video Unexpected license problem; exiting. Hit return to exit If this happens hit the Enter key to close Ansys, do not click the X in the corner.
  8. Reopen Ansys to refresh the license and the licensing is complete.

Personal Machine Use (Applies to Faculty/Staff/Cadets)‚Äã

PREREQUISITE

Please be aware the you will need around 250 GB of free space to install the full suite of ANSYS

  • Compressed Installer Packages - 52 GB
  • Extracted Installer Files - 50GB
  • Full Ansys install is around 145 GB e ISO downloaded files

1. Download ISO Installer Packages‚Äã

Download all the files found within the Ansys 2025 R1 folder Download Link (Click the 3 dots then click download)

This will download all the files in a .zip file named Ansys 2025 R1.zip and will include the following files:

  • Service Pack 3 Folder -> ANSYS_2025R1.03_WINX64.zip (11.3 gb)
  • 7z2501-x64.exe (1.6 mb)
  • ANSYS2025R1_WINX64.zip.001 (23.5 gb)
  • ANSYS2025R1_WINX64.zip.002 (23.5 gb)
  • ANSYS2025R1_WINX64.zip.003 (7.9 gb)

This is a very large file and may take a long time to download

2. Extract Ansys 2025 R1.zip‚Äã

Once Ansys 2025 R1.zip has completed navigate to where you downloaded it to and then extract all the contents.

3. Install 7zip‚Äã

Run 7z2501-x64.exe and click install, then close once complete.

3. Extract the Installer Files from the Compressed Installer Packages‚Äã

TIP

To select multiple files you can click the first file then hold CTRL and click the additional files you want to select as well

  1. Select/highlight all the following files:

    • ANSYS2025R1_WINX64.zip.001 (23.5 gb)
    • ANSYS2025R1_WINX64.zip.002 (23.5 gb)
    • ANSYS2025R1_WINX64.zip.003 (7.9 gb)

  2. Once all of those files are selected/highlighted, right click and hold while dragging to an empty space in the directory.

    • A popup will come up and you will see 7-zip in the menu, hover over that and then a new menu will show up to the right of it. Select Extract here
    • This will create a folder named ANSYS2025R1_WINX64.

4. Install ANSYS 2025 R1‚Äã

REMINDER
  • This is a very lengthy install, and can sometimes take over an hour to two hours depending on your hardware and package selections.
  • If the installer looks like it is not progressing please let it continue.
  • It is recommended to just let the installer go and not use your computer during the install process.
  1. Navigate to the ANSYS2025R1_WINX64 folder and run setup.exe
  2. Click Install Ansys Products
  3. Under the Legal section: Read the license agreement and SELECT I agree to the terms of the licence agreement, including, to the extent applicable, the cross-border transfer of data and collection of geographic location data under Section 14 and then click Next
  4. Under the Desti nation section: Keep the installation destination default and keep the Associate file extensions with Ansys products defaulted. Click Next
  5. Under the Products section: Click Next unless you want to only install specific programs
  6. Under the Options section: Keep Yes, automatically configure my interfaces defaulted and click Next
    • Click Next for the next few pages (keep the options default selections) until you reach the confirmation section.
  7. Once you get to the Confirmation section, verify your install options then click Next to begin the install.
    • You can click the View progress log to see the status of what is installing
    • A firewall will popup up asking to allow networks access to the app Electronics Desktop 2025 R1, click Allow
  8. You will see Click 'Next' to continue when the install finishes. Click Next

5. Enable Web Shared Licensing‚Äã

  1. Upon exiting the installer the Ansys Licensing Client settings will open. (If it does not you can run it manually by navigating to the Start Menu -> All -> Ansys 2025 R1 -> Ansys Licensing Settings 2025 R1)
  2. Click on the left hand side menu tab Products under Web Licensing
    • Click the slider to the right of Enabled to enable the web licensing
    • A firewall will popup up asking to allow networks access to the app Ansys Licensing Settings. Modified at Ansys Release 2025 R1, click Allow
  3. You should see in the background an internet browser popup with the Ansys login portal.
    • Input your Ansys ID Email in the Email adress text field from when you created your Ansys account from the instructions above and click Continue
    • Input your Ansys ID Password in the Password text field from when you created your Ansys account from the instructions above and click Continue
    • Once you see You are now signed into your Ansys account. You can close this tab and continue using the application you can close the internet browser, and you should see the Web Licensing is enabled. You can now close the Ansys Licensing Settings 2025 R1 app.

6. Install the Service Pack 3 Update‚Äã

  1. RESTART YOUR COMPUTER AT THIS POINT BEFORE YOU INSTALL THE SERVICE PACK 3
  2. Navigate to the folder Service Pack 3 and extract the contents of ANSYS_2025R1.03_WINX64.zip into the same folder.
  3. Right click setup.exe and select Run as Aministrator
  4. Click Next through all the prompts (leave defaults) until the installer starts.
  5. You will see Click 'Next' to continue when the install finishes. Click Next
  6. Click Exit to exit the installer

7. Cleanup Files‚Äã

At this point you can delete all the installer files and the extracted installation files to help clear up space

8. Verify Installation and License‚Äã

  1. You can verify the Installation and Licensing is working properly by opening any Ansys program (Fluent is used in the video example).
    • If you encounter any firewall popups, just allow them all through the network as seen in the video.

Support

If you are having issues with Ansys please use the academic forums as that is the only support available for Academic Programs (you must sign in with your Ansys ID Account): https://innovationspace.ansys.com/forum/
