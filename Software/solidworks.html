Important Prerequisites#

IMPORTANT
  • Please ready instructions carefully
  • This is for Windows Operating Systems only. OSX (Mac) is NOT officially supported by Solidworks
  • If you have any previous versions of Solidworks installed please see below as you MUST uninstall any prior versions of Solidworks before proceeding.
  • If you DO NOT have previous version of Solidworks installed or are doing a fresh install please skip the uninstall steps

Uninstall Previous Solidworks Versions#

Please note the following when uninstalling: backup all previous Solidworks Data you want to keep, and note that you can choose to perform a standard, custom, or complete uninstall.

The standard uninstall removes Program Files and Folders associated with Solidworks products. By default, all Solidworks products for the selected release are specified for a standard uninstall.
A custom uninstall allows you to choose and removes one or more of the following: Program Files and Folders, registry keys and data folders (Solidworks Toolbox & files and folders from the original download locations..)
[RECOMMENDED] A complete uninstall removes the installatoin directories, registry keys (expired serial numbers), and data folders

Uninstall Directions#

  1. Press the Windows Key (âŠž) + R to bring up the run command prompt/textfield
  2. Type "appwiz.cpl" and hit enter to bring up the "Programs and features" to uninstall software. Scroll down and select the previous version(s) of Solidworks you are uninstalling
  3. On the uninstall screen, ensure that the product(s) you want to uninstall is selected (If you uninstall all products, the Solidworks Installation Manager is also uninstalled
  4. Clear the products you do not want to remove, if you retain any product(s), Solidworks Installation Manager is not removed. Choose whether to perform a standard, custom, or complete uninstall (Complete uninstalls are recommended just make sure you have any Solidworks data backed up beforehand)
  5. Click change in advanced options, the Advanced Options screen appears. The program Files and Folders and Folders option is checked by default. Select all options to perform a complete uninstall
  6. Click Back to Summary
  7. On the Summary screen, expand Advanced Options to display the uninstall method and entries
  8. Click Remove Items

Installation Instructions#

Request SEK Number
  • As of 2021/2022 release you will need an SEK Number isntead of an SDK-ID (Used in previous years) to install Solidworks.
  • Please access the shared Box Link to get SEK Number
  • This license is good for up to one year, each year you will need to get a new license.

01 - Navigate to Downlaod Site#

  1. Goto the SOLIDWORKS Community Download site: https://www.solidworks.com/support/community-download#no-back. Fill out all the information under Contact Information and make sure to use your VMI email. Under Product Information Select "Yes" for "I already have a Serial Number that starts with 9020. Select the newest version under Academic Version then click Request Download.

02 - Terms and Conditions#


2. This will bring you to the terms and conditions page, please read and click Accept and Continue.

03 - Download Installer#


3. The next page will take you to SOLIDWORKS Community Access. Click the Download button to download the SolidWorksSetup.exe file then run it to allow it to self extract to start the SOLIDWORKS Installer.

04 - Reboot Pending?#


4. If you see this message at the start of the setup, please restart your computer then restart the setup.

05 - Installation Specification#


5. Select "Install on this computer" and click next.

06 - Serial Number#

IMPORTANT
  1. Please input the SEK number that you accessed at the start of these instructions in the green Tip Box into the Text Fields. You can not proceed without this number. (Do not use the one in the screenshot as it is a dummy placeholder serial number and will not work).

07 - Summary#


7. Read and Check the "I accept the terms of the SOLIDWORKS License Agreement, then click Download and Install

08 - Download and Install#

Recommendation

It is highly recommended to be physically plugged in with an ethernet cord during this step as WiFi connections can drop and will require starting this process over again.

  1. Duering Please stay connected to the internet as the setup is downloading and installing components at the same time. The time is dependent on your internet speeds. Do not disconnect from the internet during this installation. Once complete if it asks to restart/reboot, then restart/reboot the computer then open Solidworks and verify it opens and works properly. You may also get an activate product popup. Allow Solidworks to activate over the internet.

Support#

If you are encountering any errors during installation or have question please see support options
