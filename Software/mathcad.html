Download and Installation‚Äã

Step 1: Download Instructions

  • Login with your VMI CREDENTIALS not a personal Box Account
  • BOX DOWNLOAD LINK
  • Extract the contents to a location on your computer

Step 2: Silent Installation Instructions (RECOMMENDED INSTALL METHOD)

This silent install method will install everything in the background with no user interaction. Please allow the black cmd.exe window to close on its own even after the progress bar finishes.

*NOTE* Right click the installer.bat file and click 'Run as administrator'

Once installation completes you must license your product using the licensing instructions found below

Step 2: Manual Installation Instructions (USE IF HAVING ISSUES WITH SILENT INSTALLATION METHOD)

  1. Run setup.exe

  2. Select Install new software and click Next

  3. Read through the Software License Agreement and select if you accept or decline the agreements and click Next (You may see a popup that states Please wait while we retrieve available application information, if so please wait until you see the next step)

  4. Leave all selection defaults on the Application Selection screen and verify PTC Mathcad Prime is selected and click Next

  5. On the Customize Selected Applications screen select XMCD MCD Converter and click Install

  6. Once installation is complete click Finish

Once installation completes you must license your product using the licensing instructions found below

Step 2: Licensing Instructions

  1. Open PTC Mathcad Prime 8.0.0. While opening a new window will popup named PTC Mathcad Prime 8.0.0.0 License Wizard and select Activate Purchased License and click Next

  2. If you do not have a PTC.COM account select I am new to PTC.COM and need to create an account and click Next. If you already have a PTC.COM account please skip to step 4.

  3. Fill out the required information to create your account. This will bring you back to the Login selection page.

  4. Select I have a PTC.COM account and enter your credentials and click Next.

  5. Input your Product Code you were sent in your email. Do not change the Host ID Click Next.

  6. Make sure PTC Mathcad Prime 8.0.0.0 is selected and click Acquire License. Once the license is acquired you can exit the license wizard and Mathcad licensing is complete.

Installation
Please Note

These instructions are provided on the next page after you sign up

  1. Save the installation file to your computer (it slightly less than 300 MB, so it may take a few minutes for the download to complete depending on your Internet connection and computer performance)

  2. Right-click the Mathcad 15_EN.zip icon and choose Extract All. Select a destination for the folder that you can easily locate (such as your desktop). This will create a folder at the specified destination called "Mathcad 15_EN"

  3. Open the Mathcad 15_EN folder and double-click the "setup.exe" file to start the installation

  4. Follow the instructions and on the PTC Mathcad License Type page, choose "Acquire new license" and continue with the installation

  5. At the end of the installation process, you will be guided to set up a license. Choose "I want to acquire a license via the Internet" and "I want a trial license" on the first two pages of the PTC Mathcad License Setup. You will need the trial license installed and activated to perform any worksheet conversions to Prime 3.
