Installation
Please Note

These instructions are provided on the next page after you sign up

  1. Save the installation file to your computer (it slightly less than 300 MB, so it may take a few minutes for the download to complete depending on your Internet connection and computer performance)

  2. Right-click the Mathcad 15_EN.zip icon and choose Extract All. Select a destination for the folder that you can easily locate (such as your desktop). This will create a folder at the specified destination called "Mathcad 15_EN"

  3. Open the Mathcad 15_EN folder and double-click the "setup.exe" file to start the installation

  4. Follow the instructions and on the PTC Mathcad License Type page, choose "Acquire new license" and continue with the installation

  5. At the end of the installation process, you will be guided to set up a license. Choose "I want to acquire a license via the Internet" and "I want a trial license" on the first two pages of the PTC Mathcad License Setup. You will need the trial license installed and activated to perform any worksheet conversions to Prime 3.

Installation Quick Overview:

  • INTRODUCTION: Keep default "Install New Software" and click next
  • SOFTWARE AGREEMENT: Accept the software license agreement and click next
  • APPLICATION SELECTION: Keep the default application selections and click install
  • INSTALLATION: Wait until installation is complete, you will see progress indicators for estimate time left
  • FINISH: Click Finish to exit the PTC Installation Assistant
